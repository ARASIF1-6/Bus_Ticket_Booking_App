## 📌 Project Overview

The **Bus Ticket Booking App** is a web-based system designed to simplify the process of booking bus tickets online.  
It enables users to search available buses, select seats, and confirm bookings efficiently.

Such systems typically include features like seat selection, booking management, and admin control for routes and schedules :contentReference[oaicite:0]{index=0}.

## ✨ Features

### 👤 User Features
- 🔐 User Registration & Login
- 🔎 Search buses by route & date
- 💺 Seat selection system
- 🎫 Ticket booking & confirmation
- 📄 View booking history
- ❌ Cancel bookings

### 🛠️ Admin Features
- ➕ Add/Edit/Delete buses
- 🗺️ Manage routes & schedules
- 👥 Manage users
- 📊 Booking management dashboard
